How to Open a File From File Explorer


  1. Look for the File Explorericon on your computer. It should either be located on your desktop or on the navigation bar at the bottom of your screen. Click it open.
  2. If you know what folder your file is located in, you can now click through the directories to find your folder.
  3. If you don't know what folder it is in you can go up to the search bar in the top right corner and type in the file name.
  4. You can then press either Enteror the little arrow next to your words.
  5. This will result in a list of results. Double click the file open you want.
  6. Congrats! You can now edit and do anything you want to your document in Word.
