How to Use Word


  1. Open up the file you want to edit or open up a blank document. You can get a new blank document by going to file and then selecting a new document.
  2. We are going to list some of the basic functions of Microsoft word. Below is what some of the buttons in the font section of the home tab can do.
    1. The button that has words in it which usually defaults to Calibri (Body) has a drop down where you can select a different font.
    2. Next to the font selector tool is a box with a number in it. You can click the drop down button to choose the number of sizes you want.
    3. Under the font selector box is a B that looks bolder than other items. You can highlight the words you want to change to bold and then click that button, or click the button and continue typing in bold.
    4. Next to the bold button is a I on a diagonal. This is the italicized button. You can highlight words and click the button to turn it italicized. You can also click the button and keep typing which will be italicized.
    5. Next to the bold button is the underline button. It looks like a U with a line underneath. Similarly to the other buttons you can highlight words to underline or press the button and keep texting in underline words.
    6. There is a button with a capital letter A with a red rectangle underneath it. You can click the drop down box to choose a different color of text. You can highlight a section of text and then choose a color to change the color, or click a new color to keep typing in those colors.
  3. The next important thing to do is to click the review tab.
    1. You can click the editor button on the left that will do spell and grammar checks on your entire document.
    2. The next item is a word count button that is a few below the editor button. This will tell you the word count of the entire document or highlighted section
    3. There is also the button that says read aloud which is in the section over from the editor and word count button. This when pressed will read your entire document to you out loud.
  4. Finally, you will want to save your document. Click the file button on the top right of the toolbar.
  5. Click the save button that is about halfway down the side bar.
  6. This will open your file directory. Choose the folder you want to save this file into.
  7. Type in the file name on the file name input box
  8. Click save which is next to the file name input box
  9. Congratulations! You just created and saved a word document.
