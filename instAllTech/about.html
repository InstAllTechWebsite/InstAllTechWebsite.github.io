InstAllTech

About the Team

instAllTech is a single website for non-technical users to find tutorials and resources on installing and using various apps and software. Developed as part of a Human-Computer Interaction Capstone project, instAllTech focuses on addressing the difficult conversation of getting tech help from a distance.

InstAllTech was created by: Ruipu Hu, Angela Ngo, Megan Rutch, and Lucas Soohoo

Interested in the code? Check out our Github Repo!
