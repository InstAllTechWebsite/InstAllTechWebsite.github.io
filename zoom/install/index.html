How to Install Zoom


  1. Go to https://zoom.us/download
  2. Click on the Download> button under Zoom Client for Meetings.
  3. Open the ZoomInstaller.exe file from the downloads list at the bottom left corner of your window to install it on your computer.
  4. If you get any pop-ups requesting permissions from your computer, click Allow.
  5. Wait for the installation to finish. Once it's done, you are ready to use Zoom!
